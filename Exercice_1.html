  3. Le webs est constitue d'un client et d'un serveur qui a'echangent l'un de l'autre des services le client est accessible au navigateur web qui demende au requete au serveur, et le serveur a son tours ayant dans son contenu le programme web fournissant la reponse a la requete.

    -> Lorsau'on consulte un site web, le client saisit l'adresse dans ke navigateur , le navigateur consulte le DNS pour trouver l'adresse reelle du site web, le serveur web traite l'URL et donne le controle au back-end. Il genere la page HTML et transmet au serveur web. Par le HTTP le serveur envoit la page au navigateur.
  4. Les technologies utilisees dans le web sont:
    • Le navigateur
    • HTML
    • CSS
    • JAVA SCRIPT
  5. Le navigateur est un logiciel concu pour consulter et afficher des pages webs . Le client saisit l'URL du site web , le serveur recherche la page correspondante et le navigateur consulte et affiche la page.
