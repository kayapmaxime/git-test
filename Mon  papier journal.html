    HTML qui veut dire(HyperText Markup Language) est un langage de balisage concu pour representer les pages web base sur un standart Generalized Markup Language developper par WorldWide Web.
    Pour ecrire un programe en HTML, il faut inserer des elements et des attributs dans des balises en uilisant la syntaxe imposee. Par exemple un un paragraphe de texte est ecrit dans une balise (balise ouvrante et balise fermente) puis sera afficher sur le net avec un navigateur.

    CSS a son tour signifie (Cascading Style Sheet), forment un langage informatique qui est utiliser pour apporter beaucoup de style a notre site, qui est aussi charger de l'emplacement de tout les constituants de notre page web , contient aussi plusieurs balises
